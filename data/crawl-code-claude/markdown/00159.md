よくある質問
### Claude Enterprise planとはどのようなものですか？
Claudeは、信頼性が高く安全で、協働的なAIエキスパートであり、組織の知識とワークフローに統合し、高品質な作業を支援します。Claudeは、組織内のさまざまなビジネス機能で生産性と創造性を向上させます。Enterprise planは、大規模な知識のアップロード、強化されたセキュリティとユーザー管理、広範にわたる作業をサポートするために部門横断的なチーム全体で拡張できるAIソリューションを必要とする組織向けに設計されています。
### Claudeを自社の製品やサービスにどのように統合できますか？
ユーザー向けの体験や新製品をClaudeを使用して作成したい開発者の方は、Anthropic APIが最適です。各種APIプランの詳細については、当社の営業チームまでお問い合わせください。利用開始には、[当社の開発者向けドキュメントをご覧ください](https://docs.claude.com/en/home)。
### コネクタの利用規約はどこで確認できますか？
ディレクトリ内のコネクタは、Model Context Protocol（MCP）を使用してサードパーティの開発者によって構築され、維持されます。 各コネクタプロバイダーは、独自の規約とプライバシーポリシーを定めており、通常、接続時の OAuth 認証プロセス中に提示されます。 このディレクトリでのコネクタのレビューと公開方法に関する詳細は、[MCP ディレクトリ利用規約](https://support.claude.com/en/articles/11697081-anthropic-mcp-directory-terms-and-conditions)と[MCP ディレクトリポリシー](https://support.claude.com/en/articles/11697096-anthropic-mcp-directory-policy)をご覧ください。
