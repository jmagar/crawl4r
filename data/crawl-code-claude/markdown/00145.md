Sicher
surfen
mit
Claude
Claude bringt KI direkt in Ihren Browser. Dies ist zwar leistungsstark, birgt jedoch Risiken, die Cyberkriminelle möglicherweise ausnutzen.
Risiken besser verstehen
[Risiken besser verstehen](https://support.claude.com/en/articles/12902428-using-claude-for-chrome-safely)Risiken besser verstehen
### Mit vertrauenswürdigen Websites beginnen
Sie sollten nur bekannte Websites zulassen, solange Sie lernen, wie Claude funktioniert.
### Vertrauliche Aktionen überprüfen
Bestätigen Sie die Aktion immer, bevor Claude finanzielle, persönliche oder geschäftskritische Aufgaben erledigt.
### Auf ungewöhnliches Verhalten achten
Einige Websites verbergen möglicherweise Anweisungen, die Ihre überschreiben. Wenn Claude unerwartet reagiert, können Sie es pausieren und überprüfen.
### Probleme sofort melden
Wenn Sie merkwürdige Verhaltensweisen über die Feedback-Optionen melden, verbessern Sie die Sicherheit.

## 
Häufig
gestellte
Fragen
### Wie funktioniert Claude Code mit der Chrome-Erweiterung?
Mit Claude Code können Entwickler direkt in Chrome arbeiten und testen. Diese Integration ermöglicht eine schnellere Iteration bei browserbasierten Projekten.
### Wie funktioniert Claude Desktop mit der Chrome-Erweiterung?
Benutzer von Desktop-Apps können eine Aufgabe in [Claude Desktop](http://claude.com/download) starten und im Browser erledigen lassen, ohne das Fenster wechseln zu müssen. Dazu müssen sie Claude in Chrome einfach als Connector aktivieren. 
### Welche Aktivitäten sollten vermieden werden?
Vermeiden Sie Finanztransaktionen, Passwortverwaltung oder alles, was vertrauliche persönliche Daten beinhaltet. Starten Sie mit vertrauenswürdigen Websites und bekannten Workflows, bei denen Sie Claude zutrauen, Aktionen auszuführen. Verwenden Sie es niemals für wichtige Entscheidungen, ohne dies sorgfältig zu prüfen. Und natürlich sollten Sie sicherstellen, dass Ihre Nutzung unseren [Richtlinien zur zulässigen Nutzung](https://support.anthropic.com/en/articles/12005017-using-agents-according-to-our-usage-policy) entspricht. [Mehr erfahren](https://support.claude.com/en/articles/12902428-using-claude-for-chrome-safely)
### Wie kann ich steuern, auf welche Inhalte Claude zugreift?
Sie können Aktionen im Voraus genehmigen, die Claude auf Websites durchführen kann, bevor Sie mit der Arbeit beginnen. Sie können den Ansatz von Claude im Voraus prüfen und ihn dann ausführen lassen. Claude fragt immer noch einmal nach, bevor es bestimmte irreversible oder potenziell schädliche Aktionen ausführt, z. B. einen Kauf. Bei bekannten Workflows können Sie alle Berechtigungen überspringen, aber Sie sollten Claude genau überwachen. Obwohl es einige Sicherheitsmaßnahmen für vertrauliche Aktionen gibt, können Cyberkriminelle Claude trotzdem für bösartige Aktionen ausnutzen. [Mehr erfahren](https://support.claude.com/en/articles/12902446-claude-for-chrome-permissions-guide)
### Welche Sicherheitsrisiken bestehen dabei?
Browser-KI ist einzigartigen Sicherheitsrisiken ausgesetzt, wie etwa Prompt-Injection-Angriffen, bei denen Cyberkriminelle versuchen, Claude für böswille Aktionen auszuführen, z. B. zur Freigabe Ihrer Bankdaten oder zum Löschen wichtiger Dateien. Obwoh wir Schutzmaßnahmen implementiert haben, sind diese nicht narrensicher. Angriffsvektoren entwickeln sich ständig weiter und Claude kann Halluzinationen verursachen, die zu Aktionen führen, die nicht beabsichtigt sind. Wir haben unsere Testergebnisse veröffentlicht, einschließlich möglicher Angriffsszenarien, damit Sie fundierte Entscheidungen treffen können. Wir empfehlen Ihnen dringend, sich [über die Risiken zu informieren](https://support.claude.com/en/articles/12902428-using-claude-for-chrome-safely), bevor Sie dieses Produkt verwenden.
