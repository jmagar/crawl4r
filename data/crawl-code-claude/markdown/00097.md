선도적인 금융 운영 플랫폼 Ramp는 개발 속도로 명성이 높습니다. 기업의 지출 관리 및 수동 작업 자동화를 돕는 새로운 기능들을 거의 이틀에 한 번꼴로 출시하고 있기 때문입니다.
Ramp는 Claude Code를 통해 소프트웨어 개발 프로세스를 혁신하고 있습니다. 
Claude Code는 개발자들이 애플리케이션을 더 빠르게 제작하고, 더 효과적으로 협업하며, 고객에게 더 큰 가치를 제공할 수 있도록 지원합니다.
Claude Code를 통해 Ramp는 다음과 같은 성과를 달성했습니다.
  * AI가 제안한 100만 줄 이상의 코드, 단 30일 만에 구현
  * 엔지니어링 전반에서 주간 활성 사용률 50% 도달
  * 새로운 내부 툴로 인시던트 조사 시간을 최대 80% 단축


## Ramp의 엔지니어링, AI를 만나다
Ramp는 자사의 첨단 금융 플랫폼을 통해 기업들이 시간과 비용을 절약할 수 있도록 돕습니다. 가맹점별 제한, 자동화된 승인 워크플로우, 간소화된 비용 처리 등 지출 관리 솔루션을 제공합니다. 
엔지니어링 우선 접근 방식으로 유명한 Ramp는 이제 고객 제품 개발에 적용했던 것과 동일한 혁신적인 사고방식을 Claude Code를 활용하여 내부 개발 프로세스에 적용하고 있습니다.
Ramp의 선임 소프트웨어 엔지니어인 Austin Ray는 말했습니다. "기술은 늘 Ramp의 핵심 경쟁력이었습니다. 우리는 항상 사용 가능한 최첨단 도구를 적극적으로 탐색하고 신속하게 도입하는 문화를 만들어왔습니다. 
Claude Code를 발견했을 때, 우리 팀은 그 잠재력을 즉시 알아보고 워크플로우에 통합했습니다. 최첨단 기술을 적극적으로 도입하는 회사의 기존 비법을 그대로 적용한 것입니다"
## Ramp가 Claude Code를 채택한 이유
Ramp 엔지니어링 팀이 여러 AI 코딩 어시스턴트를 평가한 결과, Claude Code가 자사의 개발 니즈에 완벽히 부합하는 독보적 역량을 제공한다는 걸 확인했습니다. 
개별 엔지니어들의 실험으로 시작된 시도가 빠르게 전사적인 활동으로 자연스럽게 확산되었습니다.
"엔지니어들이 스스로 사용해본 결과, 그 경험이 너무 혁신적이어서 동료들에게 반드시 공유해야 한다고 느꼈습니다. 이러한 초기 경험은 조직 전체에 더 큰 열정을 불러 일으켰습니다."라고 Ray는 설명합니다.
이러한 빠른 확산은 강력한 지표를 통해 입증됩니다. 다수의 엔지니어가 Claude Code를 매일 사용하고 있으며, 주간 사용률은 거의 50%에 달합니다. 이러한 높은 유지율은 엔지니어들이 Claude Code를 적극적으로 도입하고 있으며, 그 효용성을 점차 더 많이 체감하고 있음을 보여줍니다. 
엔지니어링팀은 Claude Code의 효과적인 도구 통합 및 강력한 코드 제안 기능을 높이 평가합니다. 이 점이 다른 AI 모델과 차별화되는 부분입니다.
Ramp와 Anthropic의 협력 관계는 Claude Code의 도입 속도를 더욱 가속화하는 계기가 되었습니다. "Anthropic 팀과의 파트너십은 무척이나 탁월한 선택이었습니다." Ray는 말합니다. 
"우리는 엔지니어들이 문제가 발생했을 때 즉각적으로 알릴 수 있는 직통 채널을 보유하고 있으며, 문제 발생 시 대응은 늘 신속하고 효과적이었습니다. 이러한 수준 높은 지원 덕분에 Claude Code 제품 자체와 Anthropic과의 협력 관계 둘 다 굳건한 믿음을 가지게 됐고, 결국 우리는 이 도구의 사용 범위를 자신 있게 확대할 수 있었습니다."
## 더 생산적이고 강한 회복력을 가진 엔지니어링 조직
Ramp 엔지니어들이 Claude Code를 활용하여 더 빠르게 작업하는 과정을 설명하겠습니다.
  * **테스트 자동화** : 


Ramp는 유연한 명령줄 인터페이스를 활용해 Claude Code를 테스트 프레임워크에 연결하여 개발-테스트 루프를 간소화하는 Claude Code의 확장 기능을 개발했습니다. 
Claude Code는 기존 개발 도구처럼 터미널에서 호출되어 표준 입력/출력 스트림을 처리할 수 있습니다. 
따라서 Ramp 엔지니어들은 Claude Code와 자체 테스트 프레임워크 간에 입력 및 출력을 손쉽게 연결하여, 코딩, 테스트 생성, 오류 수정과 같은 반복 작업을 자동화하는 맞춤형 추상화 구조를 만들 수 있습니다. 
Claude Code는 테스트 실패 상황을 독립적으로 분석하고, 필요한 코드를 조정하며, 모든 것이 통과할 때까지 테스트를 다시 실행합니다. 이 모든 작업은 기존 개발 환경과 원활하게 연동되는 익숙한 인터페이스를 통해 수행됩니다.
  * **향상된 문서화** : 


Claude Code는 Ramp 코드베이스 전체에 걸쳐 종합적인 문서를 자동으로 생성하며, 현재는 물론 미래의 개발 과정에도 유용한 일관된 상황별 정보를 제공해 문서 작성 과정을 개선합니다.
  * **병렬 개발 워크플로우** : 


개발자는 동일한 코드베이스 내에서 여러 Claude Code 세션을 동시에 실행하며, 각 세션은 서로 다른 작업을 처리합니다. 이러한 병렬 처리 방식은 대기 시간을 줄이고 처리량을 크게 증가시킵니다.
  * **AI 기반 인시던트 대응** : 


Ramp는 Claude Code를 Datadog 및 Sentry와 같은 관측 가능성 스택에 연결하여 인시던트 관리를 위한 전문 도구를 개발하고 있습니다. 
Claude Code는 모델 컨텍스트 프로토콜 (MCP) 서버와 통합되어 인시던트 발생 시 로그, 오류 보고서, 시스템 메트릭을 자동으로 수집하며, 엔지니어에게 통합된 관련 정보를 신속하게 제공합니다. 초기 관찰 결과를 보면 최초 인시던트 분류 시간을 최대 80%까지 단축할 수 있을 것으로 예상됩니다. 
이 통합 덕분에 온콜 엔지니어는 여러 출처의 데이터를 수동으로 취합할 필요 없이 복잡한 시스템 문제를 신속하게 파악할 수 있기 때문입니다.
  * **티켓 투 코드 자동화** : 


엔지니어들은 프로젝트 관리 시스템을 Claude Code에 직접 연결하여 수동으로 정보를 전송하지 않고도 티켓 컨텍스트를 가져오고, 요구 사항을 이해하며, 솔루션을 구현할 수 있습니다.
## 팀 전반의 AI 실험 지원
Ramp의 엔지니어들은 Claude Code를 개발 실무에 깊이 통합해 왔으며, AI가 제안한 코드를 매달 100만 줄 이상 처리하는 등, 조직 전반에서 여러 팀들이 특수 워크플로우에 Claude Code를 도입하고 있습니다. 
"Claude Code는 제품, 디자인, 데이터 팀은 물론 엔지니어링 전반에서 사용되고 있습니다." Ramp의 엔지니어인 Zack Field는 말했습니다. 
"데이터 사용 사례는 특히 흥미롭습니다. 팀원들이 복잡한 쿼리를 작성하는 대신 자연어로 Snowflake 데이터 웨어하우스와 상호작용할 수 있게 해주기 때문입니다."
이 자연어 인터페이스는 기술과는 전혀 연관이 없는 영업, 리스크, 회계, 재무, 채용 분야의 직원이 전문적인 코딩 지식 없이도 회사 데이터에서 직접 가치 있는 통찰력을 추출할 수 있도록 돕습니다.
## Ramp의 장기적 비전에 부합하는 AI
Ramp는 단순히 엔지니어들의 작업 속도를 높이는 데 그치지 않고, 조직 전체의 프로세스 변화를 통해 자신들의 AI 도구 효율성을 최적화하는 데 주력하고 있습니다. 
이러한 접근 방식은 엔지니어와 AI 시스템이 함께 발전하게 되어, 시간이 지남에 따라 더 많은 이점을 창출하는 선순환을 만들어냅니다.
Ramp는 빌딩의 미래가 가져올 변화와 AI 역량이 지속적으로 발전함에 따라 소프트웨어 엔지니어링 환경이 어떻게 진화할지에 대해 기대하고 있습니다. 이미 Claude Code 는 엔지니어들이 갖고 있는 기존 기술을 최대한 활용할 수 있도록 지원하며, 시간과 비용을 절약하는 제품 제작 및 출시라는 중요한 업무에 집중할 수 있도록 지원합니다. 
Ramp는 엔지니어들이 고급 아키텍처와 혁신, 고객 중심적 문제 해결에 점점 더 집중하고, AI가 더 많은 구현 세부 사항을 처리할 수 있는 미래를 예상합니다. 
현재 이러한 워크플로우를 선도하는 Ramp는 AI 기반 개발 혁신의 최전선에 서있으며, 팀과 고객 모두에게 이익을 주는 새로운 가능성을 창출하고 있습니다.
