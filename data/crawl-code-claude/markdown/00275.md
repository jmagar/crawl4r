# What is the Enterprise plan?
Updated today

The Enterprise plan is designed for organizations that require advanced capabilities, enhanced security, and scalable solutions to empower every employee with AI. This plan builds upon the features of [our Team plan](https://support.claude.com/en/articles/9266767-what-is-the-team-plan), providing a robust set of tools and resources to meet the complex needs of your business.
## What‚Äôs included in the Enterprise plan?
The Enterprise plan supports deep, cross-functional workflows and includes everything in the Team plan in addition to the following features:
  * **Enterprise-grade security features** to ensure the safety and compliance of your organization's data including:
    * **Single-Sign-On (SSO) and Domain Capture:** Enhance authentication security and user experience.
    * **Just-in-Time Provisioning (JIT)** : Streamline user authentication and manage access across your organization.
    * **Role based permissioning:** Customize access levels and control user privileges across your organization.
    * **Audit Logs:** Capture key information about user actions, system events, and data access.
    * **System for Cross-domain Identity Management (SCIM)**
    * **Custom data retention controls**
    * **Compliance API:** Programmatically access Claude usage data including activity logs, chat histories, and file content, with filtering capabilities by user and time range.
  * **Enhanced context window** that enables users to upload hundreds of sales transcripts, dozens of 100+ page documents, and 100K lines of code.
    * When using Claude Sonnet 4.5, there is a 500k context window when chatting with Claude and a 1M context window for Claude Code.
    * The standard 200k context window is available when chatting with Claude Opus 4.1, Opus 4, or Sonnet 4.
  * **Increased usage** , which means more messages with Claude.
  * **Native integrations with data sources like GitHub** provide the ability for engineering teams to brainstorm alongside your codebase, iterate on new features, onboard engineers, and debug issues.


## Seat types for Enterprise plans
There are two different seat types available to purchase and allocate to team members:
  * **Standard seats** include all the core Enterprise features listed above.
  * **Premium seats** include access to Claude Code and higher usage in addition to the features listed above.
    * See [Using Claude Code with your Team or Enterprise plan](https://support.claude.com/en/articles/11845131-using-claude-code-with-your-team-or-enterprise-plan) to learn how to access Claude Code with your premium seat.


Refer to this article for general information about premium seats: [How to purchase and manage premium seats](https://support.claude.com/en/articles/12004354-how-to-purchase-and-manage-premium-seats).
## How can I get the Enterprise plan for my organization?
If you're interested in leveraging the advanced capabilities of the Enterprise plan for your organization, there are two different ways to gain access:
  1. Reach out to our Sales team via the [Contact Sales form](https://claude.com/contact-sales).
  2. Visit the [AWS Marketplace](https://aws.amazon.com/marketplace/pp/prodview-nnvi6wff6ef6m)
