# 
Claudeをデスクトップに配置
Claudeはバックグラウンドで常駐し、必要なときにすぐに使えて、不使用時にも決して邪魔になりません。

### いつでもアクセス可能
任意のアプリからウィンドウを切り替えたり集中力を切らすことなくClaudeを起動できます。
### 作業を共有
スクリーンショットの撮影、ウィンドウの共有（Macのみ）、チャットへのファイルの直接ドラッグにより、自分が見ている画面をClaudeに表示できます。
### 声に出しながら考える
音声入力に切り替えてブレインストーミング、複雑な問題を解決、または考えながら下書きの作成ができます。（Macのみ）
### お使いのツールと提携
Claudeは、お使いのコードエディター、ローカルファイル、データベース、Claude in Chromeと直接連携し、問題のデバッグ、コードのテスト、タスクの自動化を行います。

## 
よくある質問
### デスクトップアプリまたはモバイルアプリを使用するには、有料プランが必要ですか？
いいえ。ClaudeアプリはFree、Pro、Max、Team、Enterpriseのすべてのプランタイプで利用できます。一部の機能は、有料プランでのみ利用できます。
### デスクトップアプリとブラウザでClaudeを使用する場合の違いは何ですか？
デスクトップアプリは、ドックから常にアクセスでき、システム内のどこからでも即座に利用できるクイックエントリーが含まれています。Claudeをローカル環境で動作するツールやファイルと連携させるためのデスクトップ拡張機能も利用できます。
### デスクトップ、ウェブ、モバイルで同じアカウントを使用できますか？
はい。ログインしている間、会話、プロジェクト、メモリ、環境設定がすべてのデバイス間で同期されます。
### Claudeを使用中のツールと接続するにはどのようにすればよいですか？
コネクタを使用すると、Claudeはアプリやデータベースなどのツールと連携できます。GoogleドライブやSlackなどのウェブベースのコネクタは、ウェブ、デスクトップ、モバイルなど、Claudeを使用しているあらゆる場所で機能します。モバイルでは、ClaudeはマップやiMessageなどのデバイス固有のアプリにも接続できます。デスクトップアプリは、ローカルにインストールされたデスクトップ拡張機能もサポートしています。これは、ローカルファイルシステム、ブラウザ、ネイティブアプリケーションへのアクセスを提供します。
### Claude DesktopはChrome拡張機能とどのように連携しますか？
Claude in Chromeはコネクタとして含まれており、有効化するとすぐに使用できます。どんな会話でもこの機能を有効にすれば、Claudeはデスクトップアプリからブラウザ上でナビゲーション、クリック操作、フォームへの入力ができます。 ウィンドウを切り替えることなくタスクを開始します。 [Claude in Chrome](http://claude.com/chrome)の詳細はこちら。
### デスクトップアプリを社内の組織に導入するにはどうすればよいですか？
Claudeデスクトップアプリは、MSIXインストーラー（Windows）とPKGインストーラー（Mac）による標準的なエンタープライズ展開をサポートしています。バージョン更新を管理し、自身のスケジュールに合わせてでテストと承認を行えます。管理対象デバイスの自動SSO認証を取得し、チームがアクセスできるデスクトップ拡張機能を事前承認することも可能です。
### どこでサポート受けたり、詳細を確認できたりしますか？
セットアップガイド、トラブルシューティング、システム要件については、[ヘルプセンターにアクセスしてください](https://support.claude.com/en/)。
