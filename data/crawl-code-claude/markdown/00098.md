Rampは、金融業務プラットフォーム業界のリーダーとして、その開発スピードに定評があり、企業の支出管理や手動作業の自動化に役立つ新機能をほぼ隔日でリリースしています。
Rampは、Claude Codeを活用してソフトウェア開発プロセスを変革しており、開発者はアプリケーションをより迅速に構築し、より効果的に協業し、顧客により大きな価値を提供しています。
Claude Codeを活用したことで、Rampは以下を達成しました。
  * AI提案コードに実装行数がわずか30日間で100万行超
  * エンジニアリング部門全体で週間アクティブ利用率が50％到達
  * 新社内ツールの導入で、インシデント調査時間を最大80%短縮


## RampにおけるエンジニアリングとAIの融合
Rampは、同社の高度な金融プラットフォームを通じて企業の時間とコスト削減を支援し、加盟店レベルでの制限設定、承認ワークフローの自動化、経費処理の効率化を実現する支出管理ソリューションを提供しています。 Rampは、エンジニアリングファーストのアプローチで知られており、現在、顧客向け製品開発を推進する革新的な考え方を、Claude Codeを使用して社内の開発プロセスにも適用しています。
RampのシニアソフトウェアエンジニアであるAustin Ray氏は次のように述べています。「テクノロジーは常にRampにとっての競争優位性でした。当社は常に利用可能な最も先進的なツールを探求し、迅速に採用する文化を築いてきました。Claude Codeを発見した際、当社のチームはすぐにその可能性を認識し、ワークフローに統合しました。最先端技術を取り入れるという当社の伝統を継承したのです」
## RampがClaude Codeを採用した理由
複数のAIコーディングアシスタントを評価した結果、Rampのエンジニアリングチームは、Claude Codeが自分たちの開発ニーズに合致した独自の機能を提供することを確認しました。当初は、個々の実験にすぎませんでしたが、すぐに草の根的な活動へと発展しました。
「エンジニアたちは自ら試してみると、非常に革新的な体験をし、同僚と共有せずにはいられないと感じられたのです。この最初の体験が熱意を生み出し、組織全体に広がりました」とRay氏は説明します。
この急速な普及は、説得力のある指標によって裏付けられています。つまり、多くのエンジニアが毎日Claude Codeを使用しており、約50%が週に1度は利用しています。この高い継続利用率は、エンジニアがますますClaude Codを活用し、その恩恵を受けていることを示しています。同社のチームは、Claude Codeの効果的なツール統合と強力なコード提案を高く評価しており、これがが他のAIモデルとの差別化要因となっています。
RampとAnthropicとの関係は、導入をさらに加速させました。「Anthropicチームとの連携は特別なものです」とRay氏は説明します。「問題が発生したときにエンジニアが直接報告できるコミュニケーションチャネルが提供されており、その対応は常に迅速かつ効果的でした。このレベルのサポートにより、Claude CodeおよびAnthropicとの関係に対する信頼感が築かれ、自信を持ってこのツールの使用を拡大することが一層容易になりました」
## より生産的かつレジリエントなエンジニアリング組織
RampのエンジニアがClaude Codeを活用してどのようにスピードアップを実現しているか以下にご紹介します。
  * **テストの自動化** ：Rampでは、柔軟なコマンドラインインターフェイスを通じてClaude Codeを同社のテストフレームワークに接続する拡張機能を開発し、開発とテストのループを効率化しています。Claude Codeはターミナルから呼び出すことができ、従来の開発者ツールと同様に標準入出力ストリームを処理できます。これにより、Rampのエンジニアは、同社のテストフレームワーク間の入出力を容易にパイプ処理し、コードの記述、テストの作成、エラーの修正という反復サイクルを自動化するカスタム抽象化を作成できます。 Claude Codeはテストの失敗を独自に分析し、必要なコード調整を行い、すべてが合格するまでテストを再実行します。これらはすべて、既存の開発環境とスムーズに統合さえれた使い慣れたインターフェイスを通じて行われます。
  * **ドキュメントの強化** ：Claude Codeは、Rampのコードベース全体にわたる包括的なドキュメントを自動生成し、現在と将来の開発の両方に役立つ一貫性のあるコンテキスト情報を使用してドキュメント生成処理を改善します。
  * **並行開発ワークフロー：** 開発者は同じコードベース上で複数のClaude Codeセッションを同時に実行し、それぞれが異なるタスクを処理します。この並行処理により待機時間が解消され、スループットが大幅に向上します。
  * **AIを活用したインシデント対応** : Rampは、Claude CodeをDatadogやSentryなど可観測性スタックに接続することで、インシデント管理に特化したツールを構築しています。Claude Codeは、Model Context Protocol（MCP）サーバーと統合され、インシデント発生時に自律的にログ、エラーレポート、システムメトリックスを集約し、迅速に統合された関連情報をエンジニアに提供します。 初期の観察結果から、この統合により担当エンジニアは手動で複数の情報源からのデータを読み解くことなく、複雑なシステムの問題を迅速に理解できるため、初期のインシデントのトリアージ時間を最大80%短縮できることが示されています。
  * **チケットからのコードへの自動化** : エンジニアはプロジェクト管理システムをClaude Codeに直接接続します。これにより、チケットのコンテキストを抽出し、要件を理解し、手動での情報転送なしにソリューションを実装します。


## チーム横断でAIの実験を促進
RampのエンジニアはClaude Codeを自身の開発プラクティスに深く統合し、毎月100万行を超えるAI提案コードを処理しています。また組織内のさまざまなチームも特定のワークフローにClaude Code を採用しています。RampのエンジニアであるZack Field氏は次のように述べます。「製品チーム、設計チーム、データチーム、そしてもちろんエンジニアリングチームでの活用が進んでいます」 「特に興味深いのはデータのユースケースです。チームメンバーは、複雑なクエリを記述する代わりに自然言語でSnowflakeデータウェアハウスと対話できるようになりました」
この自然言語インターフェイスにより、営業、リスク管理、経理、財務、採用といった非技術部門のスタッフも、専門的なコーディングの知識を必要とせずに、企業データから直接価値のある洞察を引き出すことができるようになりました。
## Rampの長期ビジョンにおけるAIの位置付け
Rampは単にエンジニアの作業を迅速化を支援するだけでなく、組織全体のプロセス変更を通じてAIツールの効果を最適化することに注力しています。このアプローチにより、エンジニアとAIシステムが相互に高めあう好循環が生まれ、時間の経過に伴い相乗効果が生まれます。
Rampは、開発の未来がもたらす可能性と、AI技術の進歩に伴いソフトウェアエンジニアリングの領域がどのように進化していくかについて大きな期待を抱いています。すでにClaude Codeは、エンジニアが自身のスキルを活用できるよう支援し、時間とコストを節約する製品の構築と提供という重要な業務に専念できるようにしています。 Rampは、エンジニアがより高度なアーキテクチャ、イノベーション、顧客中心の問題解決に注力する一方で、AIが実装の詳細を扱う未来を見据えています。こうしたワークフローを先駆けて開発することにより、今日、RampはAIを活用した開発革命の最前線に自らを位置付け、チームと顧客双方に利益をもたらす新たな可能性を創出しています。
