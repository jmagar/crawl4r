Claude peut désormais utiliser les _Skills_ pour améliorer la façon d'effectuer des tâches spécifiques. Les Skills sont des dossiers contenant des instructions, des scripts et des ressources que Claude peut charger en cas de besoin.
Claude n'accède à une Skill que si elle est pertinente pour la tâche en cours. Les Skills permettent à Claude d'améliorer les tâches spécialisées, comme travailler avec Excel ou suivre les directives de marque de votre organisation.
Vous avez déjà vu des Skills à l'œuvre dans les applications Claude, où Claude les utilise pour créer des fichiers comme des feuilles de calcul et des présentations. Vous pouvez désormais créer vos propres Skills et les utiliser dans les applications Claude, Claude Code et notre API.
## Fonctionnement des Skills
Lorsqu'il travaille sur des tâches, Claude analyse les Skills disponibles pour trouver des correspondances pertinentes. Lorsqu'il en trouve, il ne charge que le minimum d'informations et de fichiers nécessaires, afin de rester rapide tout en accédant à une expertise spécialisée.
Les Skills sont :
  * **Composables** : vous pouvez combiner les Skills. Claude identifie automatiquement les Skills nécessaires et coordonne leur utilisation.
  * **Portables** : les Skills utilisent partout le même format. Créez-en une, utilisez-la dans les applications Claude, Claude Code et l'API.
  * **Efficaces** : Claude ne charge que ce qui est nécessaire, lorsque c'est nécessaire.
  * **Puissantes** : les Skills peuvent inclure du code exécutable dans les tâches où la programmation traditionnelle est plus fiable que la génération de jetons.


Considérez les Skills comme des supports d'intégration personnalisés qui vous permettent de regrouper l'expertise, faisant de Claude un spécialiste de ce qui compte le plus pour vous. Pour obtenir une description technique approfondie du modèle de conception, de l'architecture et des meilleures pratiques de développement des Skills d'agents, consultez notre [blog d'ingénierie](https://www.anthropic.com/engineering/equipping-agents-for-the-real-world-with-agent-skills).
## Les Skills fonctionnent avec tous les produits Claude
### **Applications Claude**
Les Skills sont disponibles aux utilisateurs des plans Pro, Max, Team et Enterprise. Nous fournissons des Skills pour les tâches courantes comme la création de documents, des exemples que vous pouvez personnaliser et la possibilité de créer vos propres Skills personnalisées.
![L'interface des capacités Skills de Claude.ai avec des exemples de Skills activées. ](https://cdn.prod.website-files.com/68a44d4040f98a4adf2207b6/69338018ff630211cebe56f2_690267e194f8fd4618cb330e_image.webp)
Claude invoque automatiquement les Skills pertinentes en fonction de votre tâche, sans sélection manuelle. Vous verrez même les Skills fonctionner dans la chaîne de pensée de Claude.  
  
La création de Skills est simple. La Skill « créateur de Skills » fournit un guide interactif : Claude vous pose des questions sur votre flux de travail, génère la structure de dossiers, formate le fichier Skill.md et regroupe les ressources dont vous avez besoin. Aucune modification manuelle de fichier n'est nécessaire.
Activez les Skills dans [Paramètres](https://claude.ai/redirect/website.v1.51f73c97-b077-44e7-85ba-8b27a025dfdf/settings/features). Pour les utilisateurs des plans Team et Enterprise, les administrateurs doivent d'abord activer les Skills dans l'ensemble de l'organisation.
### **Claude Developer Platform (API)**
Les Skills d'agents, que nous appelons souvent simplement « Skills », peuvent désormais être ajoutées aux demandes d'API Messages, et le nouveau point de terminaison `/v1/skills` permet aux développeurs de contrôler de manière programmatique les versions et la gestion des Skills personnalisées. Les Skills nécessitent la version bêta de l'[outil d'exécution de code](https://docs.claude.com/en/docs/agents-and-tools/tool-use/code-execution-tool), qui fournit l'environnement sécurisé nécessaire à l'exécution.
Utilisez les Skills créées par Anthropic pour permettre à Claude de lire et de générer des feuilles de calcul Excel professionnelles avec des formules, des présentations PowerPoint, des documents Word et des fichiers PDF à remplir. Les développeurs peuvent créer des Skills personnalisées pour étendre les capacités de Claude à leurs cas d'usage spécifiques.
Les développeurs peuvent également facilement créer, consulter et mettre à niveau des versions de Skills à l'aide de Claude Console.
Explorez la [documentation](https://docs.claude.com/en/docs/agents-and-tools/agent-skills/overview), notre [guide de Skills](https://github.com/anthropics/claude-cookbooks/tree/main/skills) ou [Anthropic Academy](https://www.anthropic.com/learn/build-with-claude) pour en savoir plus.
### **Claude Code**
Les Skills enrichissent Claude Code avec l'expertise et les flux de travail de votre équipe. Installez des Skills par le biais de plugins provenant de la place de marché des Skills/Anthropic. Claude les charge automatiquement lorsqu'elles sont pertinentes. Partagez vos Skills avec votre équipe grâce au contrôle de version. Vous pouvez également installer manuellement des Skills en les ajoutant dans `~/.claude/skills`. Le SDK Claude Agent fournit la même prise en charge des Skills d'agent pour la création d'agents personnalisés.

## Ensuite
Nous travaillons à simplifier les flux de travail de création de Skills et les capacités de déploiement à l'échelle de l'entreprise, afin de permettre aux organisations de distribuer plus facilement les Skills entre les équipes.
N'oubliez pas que cette fonctionnalité permet à Claude d'exécuter du code. Elle est puissante, mais cela signifie que vous devez être conscient des Skills que vous utilisez : veillez à n'utiliser que des sources fiables pour protéger vos données. [En savoir plus](https://support.claude.com/en/articles/12512180-using-skills-in-claude#h_2746475e70).
